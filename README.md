## 📌 Sobre o projeto

O Cursoruja App é um aplicativo desenvolvido em React Native com Expo, com foco em:

- Cadastro de usuários  
- Exibição de perfil com foto  
- Consumo de API (ViaCEP)  
- Uso de câmera do dispositivo  
- Context API para compartilhamento de dados  
- Navegação entre telas  

---

## 🚀 Funcionalidades

✔ Cadastro de usuário  
✔ Busca automática de endereço via CEP (API ViaCEP)  
✔ Captura de foto com a câmera  
✔ Dados globais com Context API  
✔ Tela de perfil  
✔ Tela de desenvolvedores  
✔ Navegação por abas

## ⚙️ Como instalar o projeto

1. Clonar o repositório  
git clone https://github.com/AmandhaYumi/Cursoruja-Mobile.git  

2. Acessar a pasta do projeto  
cd CURSORUJA-MOBILE  

3. Instalar dependências  
npm install  

4. Instalar Expo  
npx expo install  

Se precisar da câmera:  
npx expo install expo-camera  

5. Rodar o projeto  
npx expo start  

---

## 📱 Como usar o app

- Abrir o app no Expo Go ou emulador  
- Ir para a tela de Cadastro  
- Preencher os dados (nome, curso, disciplina, descrição)  
- Informar um CEP válido (ex: 01001000)  
- Aguardar preenchimento automático do endereço  
- Tirar uma foto com a câmera  
- Clicar em ENVIAR  
- Ver o perfil do usuário  
- Abrir tela de Desenvolvedores
