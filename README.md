## 📌 Descrição

Este projeto é um aplicativo desenvolvido em React Native que permite o cadastro de informações básicas do usuário no app Cursoruja, como:

* Nome
* Curso
* Disciplina
* Descrição pessoal

Após o preenchimento, os dados são exibidos na tela logo abaixo do formulário.

---

## 🚀 Tecnologias utilizadas

* React Native
* Expo
* JavaScript
* CSS (StyleSheet)

---

## 🎯 Funcionalidades

* Entrada de dados via formulário
* Validação de campos obrigatórios
* Exibição dos dados cadastrados

---
